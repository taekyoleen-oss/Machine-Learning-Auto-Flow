# PPT 생성 기능 사용 가이드

## 문제 해결

### 1. Stat Model이 Samples에 보이지 않는 경우

**원인**: 서버가 실행되지 않았습니다.

**해결 방법**:
```bash
# 터미널에서 서버 시작
npm run samples-server
```

서버가 실행되면 `http://localhost:3002`에서 Samples API를 제공합니다.

### 2. PPT 생성이 안 되는 경우

**원인 1**: 서버가 실행되지 않았습니다.
- 해결: `npm run samples-server` 실행

**원인 2**: Python이 설치되지 않았거나 PATH에 없습니다.
- 해결: Python 설치 확인
  ```bash
  python --version
  # 또는
  py --version
  ```

**원인 3**: 필요한 Python 패키지가 설치되지 않았습니다.
- 해결: 패키지 설치
  ```bash
  pip install python-pptx openai
  ```

### 3. 서버가 이미 실행 중인 경우

포트 3002가 이미 사용 중일 수 있습니다. 다른 터미널에서 실행 중인 서버를 확인하세요.

## 사용 방법

1. **서버 시작**:
   ```bash
   npm run samples-server
   ```

2. **브라우저에서**:
   - 모델을 작성한 후 상단 툴바의 "PPT 생성" 버튼 클릭
   - 각 모듈별 PPT가 자동으로 생성되고 다운로드됩니다

3. **생성된 파일 위치**:
   - `output/ppts/` 폴더에 저장됩니다
   - 각 모듈마다 하나의 PPT 파일이 생성됩니다

## 생성되는 PPT 내용

각 모듈별 PPT에는 다음이 포함됩니다:
- 제목 슬라이드: 모듈 이름과 타입
- 모듈 개요: AI가 생성한 모듈 설명
- 입력 데이터 설명: 각 입력 포트에 대한 설명
- 주요 변수(파라미터) 설명: 설정된 파라미터 설명
- 출력 데이터 설명: 각 출력 포트에 대한 설명
- 사용 사례: 모듈 사용 시나리오

## 디버깅

서버 로그를 확인하여 문제를 진단할 수 있습니다:
- 서버 콘솔에서 Python 스크립트 실행 로그 확인
- 브라우저 개발자 도구의 Network 탭에서 API 요청 확인
- Terminal 탭에서 에러 메시지 확인
