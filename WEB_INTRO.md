# ML Auto Flow - 웹페이지 소개

## 🎯 ML Auto Flow란?

**ML Auto Flow**는 머신러닝 워크플로우를 시각적으로 구성하고 실행할 수 있는 웹 기반 애플리케이션입니다. 복잡한 코딩 없이 드래그 앤 드롭만으로 데이터 분석부터 머신러닝 모델 구축까지 완전한 파이프라인을 만들 수 있습니다.

---

## ✨ 주요 특징

### 🎨 시각적 파이프라인 빌더
- **드래그 앤 드롭 인터페이스**: 복잡한 코드 없이 시각적으로 워크플로우 구성
- **자동 실행 순서 결정**: 모듈 간 의존성을 자동으로 분석하여 실행 순서 결정
- **실시간 결과 확인**: 각 단계의 결과를 즉시 확인하고 시각화

### 🤖 브라우저 내 실행
- **서버 불필요**: Pyodide를 활용하여 브라우저에서 직접 Python 코드 실행
- **즉시 실행**: 데이터 업로드부터 모델 훈련까지 모든 과정을 브라우저에서 처리
- **프라이버시 보호**: 데이터가 서버로 전송되지 않아 보안성 향상

### 📊 풍부한 모듈 라이브러리
**70개 이상의 모듈**로 데이터 분석의 모든 단계를 지원합니다:

#### 데이터 처리
- 데이터 로드 (CSV, Excel)
- 결측치 처리, 정규화, 인코딩
- 데이터 필터링 및 변환
- 데이터 조인 및 결합

#### 데이터 분석
- 기술 통계량 분석
- 상관관계 분석
- 이상치 탐지
- 가설 검정 및 정규성 검정

#### 머신러닝 모델
- **지도 학습**: 선형 회귀, 로지스틱 회귀, 의사결정나무, 랜덤 포레스트, 신경망, SVM, KNN 등
- **비지도 학습**: K-Means 클러스터링, PCA 차원 축소
- **통계 모델**: OLS, 포아송, 음이항 회귀 등
- **고급 모델**: 사망률 예측 모델 (Lee-Carter, CBD, APC 등)

#### 모델 평가
- 성능 지표 계산 (정확도, 정밀도, 재현율, F1, MSE, RMSE, R²)
- 혼동 행렬 시각화
- 예측 결과 분석

### 🧠 AI 기반 자동화
- **AI 파이프라인 생성**: 목표나 데이터를 설명하면 자동으로 최적의 파이프라인 생성
- **AI 데이터 분석**: 데이터를 업로드하면 자동으로 분석 계획 수립 및 실행
- **스마트 추천**: 데이터 특성에 맞는 전처리 방법 및 모델 추천

### 📈 상세한 시각화
- **통계 차트**: 히스토그램, 산점도, 박스플롯, 히트맵 등
- **모델 성능 시각화**: 학습 곡선, 혼동 행렬, ROC 곡선
- **데이터 미리보기**: 처리된 데이터를 테이블 형식으로 확인

### 💾 샘플 관리 시스템
- **샘플 저장**: 완성된 파이프라인을 샘플로 저장하여 재사용
- **샘플 공유**: 다른 사용자와 샘플 공유 가능
- **빠른 시작**: 미리 만들어진 샘플로 즉시 시작

### 📄 자동 문서화
- **PPT 자동 생성**: 각 모듈별 PowerPoint 프레젠테이션 자동 생성
- **AI 기반 설명**: 각 모듈의 기능과 결과를 AI가 자동으로 설명
- **코드 생성**: 전체 파이프라인의 Python 코드 자동 생성

## 🚀 빠른 시작

### 3단계로 시작하기

#### 1️⃣ 데이터 로드
Toolbox에서 **"Load Data"** 모듈을 Canvas로 드래그하고 CSV 파일을 선택하세요.

#### 2️⃣ 파이프라인 구성
필요한 모듈을 추가하고 연결하여 데이터 처리 및 모델 훈련 파이프라인을 구성하세요.

#### 3️⃣ 실행 및 결과 확인
"Run" 버튼을 클릭하면 자동으로 실행되고, "View Details"로 상세 결과를 확인하세요.

## 🌟 왜 ML Auto Flow인가?

### ✅ 사용하기 쉬움
- 직관적인 드래그 앤 드롭 인터페이스

- 상세한 도움말 및 가이드

- 브라우저에서 완전한 ML 파이프라인 실행
- AI 기반 자동화 기능
