## 2. 사용 목적

### 2.1 의도된 사용
- **대상 사용자**: 보험상품 개발자, 계리사, 언더라이팅 담당자
- **사용 시나리오**: [예: 50세 일본인 가입자의 5년 내 대장암 발병 확률 추정 → 진단보험금 산정]
- **출력 해석**: [예: 0-1 확률값. 0.05 미만은 저위험, 0.05~0.15는 중위험, 0.15 초과는 고위험으로 분류]

### 2.2 의도되지 않은 사용 (사용 금지 영역)
- 개인의 의료적 진단 (모델은 보험 통계 목적이며 임상 진단용 아님)
- 모집단을 벗어난 인구 (예: 80세 이상, 18세 미만)
- 일본 외 국가 직접 적용 (외부 검증 필요)

## 6. 윤리적 고려사항

### 6.1 잠재적 편향
- **수검률 편향**: 정기 검진 수검자가 모델에 과대대표 → 비수검자 적용 시 성능 저하 가능
- **사회경제적 편향**: JMDC는 대기업 근로자 중심 → 자영업자·실업자 적용 한계
- **사망 정보 편향**: 사망일 미정확 → 종단 분석 시 검열 정의 보수적 적용 필요

### 6.2 사용 시 주의사항
- 본 모델 단독으로 보험 인수·거절 결정에 사용 금지
- 자동 거절 시 반드시 인간 검토(계리사 또는 의사) 거쳐야 함
- 차등 요율 적용 시 보험감독 규정 준수 확인 필요

### 6.3 환자 권리
- 모델 추론 결과에 의한 거절 시 SHAP 기반 설명 제공 의무
- 환자의 데이터 삭제 요청 시 학습 데이터 재처리 또는 모델 재학습 검토
